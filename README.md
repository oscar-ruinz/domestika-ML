# Introducción a la IA con Python

Repositorio del curso de Domestika impartido por **Rodrigo Montemayor**, Programador.

## 📖 Descripción del Curso

Este curso de introducción a la Inteligencia Artificial con Python te permitirá adentrarte en el fascinante mundo del Machine Learning y el Deep Learning. Aprenderás a crear modelos de inteligencia artificial desde cero, comprendiendo los fundamentos teóricos y aplicándolos en proyectos prácticos con Python.

**Enlace del curso**: [Introducción a la IA con Python](https://www.domestika.org/es/courses/5239-introduccion-a-la-ia-con-python/course)

- Evaluación y optimización de modelos
